# AV2
Repositorio para entrega da atividade AV2
FRONTEND em React, utilizando NEXT, para executar o projeto siga as instruções a baixo:

Requisitos do Ambiente

Antes de rodar o projeto, certifique-se de ter instalado:

Node.js — versão 18.x ou superior

TypeScript — versão 5.x ou superior

React -

Use os comandos:

Clone o repositório:

git clone https://github.com/leodaslb/AV2

cd AV2

Instale as dependências:

npm install

E entao execute o Projeto

npm run dev

provalvemente vai estar em: 

[localHost](http://localhost:3000)
